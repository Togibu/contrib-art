# tool
Modular CLI for generating and executing scheduled Git commits to create contribution graph patterns.

## Quickstart
```bash
python tool.py init
python tool.py pattern list
python tool.py pattern choose text
python tool.py run
```

## Project layout
- `tool.py` entrypoint
- `core/` core modules
- `patterns/` pattern plugins
- `config.yml`, `patterns.yml`, `schedule.yml` generated by `init` or patterns

## Requirements
- Python 3.11+
- PyYAML

Install PyYAML:
```bash
python -m pip install pyyaml
```
